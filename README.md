# INVESTCO

Projeto Full Stack em construção...


## Rodando localmente

Clone o projeto

```bash
  git clone (https://github.com/brunaamaral95/banco-investimento.git)
```

Entre no diretório do projeto

```bash
  cd frontend
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
